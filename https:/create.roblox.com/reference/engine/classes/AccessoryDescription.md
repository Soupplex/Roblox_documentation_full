Engine Class

AccessoryDescription

Describes the appearance of an Accessory for the HumanoidDescription.

  

Summary

Properties

AccessoryType:Enum.AccessoryType
AssetId:number
Instance:Instance
IsLayered:boolean
Order:number
Position:Vector3
Puffiness:number
Rotation:Vector3
Scale:Vector3

Methods

GetAppliedInstance():Instance

Inherited Members

57 inherited from Instance

6 inherited from Object

AccessoryDescription is an object that stores the description for an Accessory. It is meant to be placed underneath a HumanoidDescription in order to work with Humanoid:ApplyDescriptionAsync().

  

API Reference

Properties

AccessoryType

Read Parallel

The Enum.AccessoryType of the Accessory referred to by this description.

AccessoryDescription.AccessoryType:Enum.AccessoryType

The Enum.AccessoryType of the Accessory referred to by this description.

AssetId

Read Parallel

The asset ID that should be applied when applying this AccessoryDescription.

AccessoryDescription.AssetId:number

The asset ID that should be applied when applying this AccessoryDescription.

Instance

Read Parallel

A reference to the Instance that should be applied when applying this AccessoryDescription.

AccessoryDescription.Instance:Instance

A reference to the Instance that should be applied when applying this AccessoryDescription. This property can be used instead of AssetId to apply accessories without uploading them to the platform.

IsLayered

Read Parallel

Whether the Accessory is layered or rigid.

AccessoryDescription.IsLayered:boolean

Whether the Accessory is layered or rigid. This is intended to be read-only. It will be corrected after calling Humanoid:ApplyDescriptionAsync() if this property doesn't match the actual Accessory contents.

Order

Read Parallel

The layered clothing sort order, if the Accessory is layered.

AccessoryDescription.Order:number

The WrapLayer.Order value when applying the Accessory, if it is layered.

Position

Read Parallel

The accessory adjustment position offset, if the Accessory is rigid.

AccessoryDescription.Position:Vector3

The accessory adjustment position offset. Only applies if the Accessory is rigid.

Puffiness

Read Parallel

The layered clothing puffiness, if the Accessory is layered.

AccessoryDescription.Puffiness:number

The WrapLayer.Puffiness value when applying the Accessory, if it is layered.

Rotation

Read Parallel

The accessory adjustment rotation offset, if the Accessory is rigid.

AccessoryDescription.Rotation:Vector3

The accessory adjustment rotation offset. Only applies if the Accessory is rigid.

Scale

Read Parallel

The accessory adjustment scale, if the Accessory is rigid.

AccessoryDescription.Scale:Vector3

The accessory adjustment scale. Only applies if the Accessory is rigid.

Methods

GetAppliedInstance

Returns the applied Accessory.

AccessoryDescription:GetAppliedInstance():Instance

Returns

Instance

Returns the applied Accessory if this AccessoryDescription is the child of an applied HumanoidDescription parented to a Humanoid.
